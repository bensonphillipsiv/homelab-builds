# Homelab Builds

This repository contains custom builds and experiments for my homelab environment.  
It focuses on **lightweight Docker image creation** primarily for in-work projects and small services, making it easy to iterate and deploy new ideas.

---

## Applications

- **the-block**  
  Custom inertial remote for triggering home automations and scenes. It uses gestures and specific orientations to activate actions and workflows within my home.
